# Eskalonado

Eskalonado static webpage/webapp using sveltekit and tailwindcss.

## Desarrollo

Primero tenemos que instalar las dependencias usando `npm install` en la raiz del directorio del proyecto, y luego arrancar el servidor de desarrollo usando:

```bash
npm run dev
```

## Development

We first need to install dependencies using `npm install` on the root of the project, and start the development server:

```bash
npm run dev
```

## Construyendo

Debemos correr el siguiente comando para que el proyecto se compile a una version de produccion minificada y optimizada para la web (carpeta 'build'):

```bash
npm run build
```

> Puedes correr de prueba la nueva version que creaste corriendo `npm run preview`.

## Building

We need to run the following command to build the project to the minified production build:

```bash
npm run build
```

> You can checkout the new version you just created running `npm run preview` after building.
